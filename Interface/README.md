# AroundU Mobile (Expo)

This folder is a React Native mobile app powered by Expo.

## Run

1. Install dependencies:
   - `npm install`
2. Start Expo:
   - `npx expo start`
3. Open on device:
   - Scan the QR with Expo Go on iOS/Android.
4. For web browser view:
   - type `w` in terminal
  

## Current scope

- Login screen only
- Local validation and mock submit state
- No backend integration yet
